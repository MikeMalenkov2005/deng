# Модель «N–P–фитопланктон» (Deng et al., 2015)

## Исходная модель

Система ОДУ:

```
ẋ = I1 − b1·x − a1·x·z/(c1+x)
ẏ = I2 − b2·y − a2·y·z/(c2+y)
ż = e1·a1·x·z/(c1+x) + e2·a2·y·z/(c2+y) − m·z
```

где **x, y** — два ресурса (биогены: азот N и фосфор P), **z** — фитопланктон.

Смысл параметров: `I1, I2` — притоки ресурсов; `b1, b2` — естественное удаление
ресурсов; `a1, a2` — интенсивности поглощения; `c1, c2` — константы полунасыщения;
`e1, e2` — коэффициенты конверсии ресурса в биомассу; `m` — смертность фитопланктона.

Модель описывает динамику концентраций азота, фосфора и фитопланктона в водоёме.

### Значения параметров модели

| Параметр | Значение |
| -------- | -------- |
| `I2`     | `0.3`    |
| `b1`     | `0.4`    |
| `b2`     | `0.5`    |
| `a1`     | `0.4`    |
| `a2`     | `0.5`    |
| `c1`     | `2`      |
| `c2`     | `3`      |
| `e1`     | `0.6`    |
| `e2`     | `0.5`    |
| `m`      | `0.1`    |

Бифуркация производится по параметру `I1`:
- Пример равновесия истребления при `I1 = 0.1`
- Пример равновесия сосуществования при `I1 = 0.3`

## Реализация на Python

### Зависимости

Для реализации используются следующие библиотеки:
numpy, scipy, matplotlib и mpl_toolkits.

Для установки всех зависимостей можно использовать команду:
`pip install -r requirements.txt`

### Запуск

Запустить код решения можно двумя способами:

1. В Jupyter Notebook:
  - Исполнить команду `jupyter notebook`
  - В ноутбуке открыть файл `solution.ipynb`
  - Запустить все ячейки файла
2. Как скрипт в Python:
  - Исполнить команду `python solution.py`

Код решения выводит вспомогательную информацию в косоль и поочерёдно строит
все необходимые по заданию графики.
